# 🛍️ Flask E-Commerce Site

A lightweight e-commerce application built with **Python**, **Flask**, and **Bootstrap 5**. This project implements a fully functional shopping cart flow, product catalog, and checkout process.

---

## 🔍 Features

- **🛍️ Product Listing**: Clean catalog view displaying available products with their prices.
- **🛒 Shopping Cart**: Dynamically add products to the cart and view items along with the calculated total amount.
- **💳 Simulated Checkout**: A smooth checkout simulation that leads to a payment success confirmation.
- **🎨 Modern UI**: Styled with Bootstrap 5 for a responsive layout.

---

## 🚀 Tech Stack

- **Backend**: Python, Flask
- **Frontend**: HTML5, Bootstrap 5, Custom CSS

---

## 🧪 How to Run Locally

### 📋 Prerequisites

Before running the application, make sure you have the following installed:
- Python 3.x
- pip (Python package installer)

### 1. Clone the repository
```bash
git clone <your-repository-url>
cd e-commerce-site
```

### 2. Set up a virtual environment (optional but recommended)
```bash
python -m venv venv
# On Windows
venv\Scripts\activate
# On macOS/Linux
source venv/bin/activate
```

### 3. Install Flask
```bash
pip install flask
```

### 4. Run the application
```bash
python app.py
```
Open your browser and navigate to `http://127.0.0.1:5000/`.
